# Análise Musical de Garoto

Este projeto realiza uma análise simbólica e acústica da obra "Jorge do Fusa", de Garoto, utilizando as bibliotecas music21 e librosa.

## Etapas:
- Extração simbólica (tonalidade, modulações, progressões)
- Análise acústica (cromagramas, MFCCs, beat tracking)
- Integração dos dados para visualizações e artigo científico
